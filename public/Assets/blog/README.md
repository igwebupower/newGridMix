# Blog Images Folder

This folder contains images for GridMix blog posts (Insights section).

## Organization

Organize your images into subfolders by topic:

- `grid-tech/` - Grid technology and infrastructure
- `renewables/` - Solar, wind, hydro, etc.
- `charts/` - Data visualizations and graphs
- `interconnectors/` - Interconnector maps and diagrams
- `policy/` - Policy-related imagery
- `analysis/` - Analysis and research visuals

## Usage

Reference images in your blog posts using:

```markdown
![Alt text](Assets/blog/your-image.jpg)
```

Or with caption:

```markdown
![Alt text|Caption text](Assets/blog/your-image.jpg)
```

See the main BLOG_IMAGES_GUIDE.md in the project root for full documentation.

## Image Guidelines

- **Format**: JPG for photos, PNG for graphics, WebP for best compression
- **Size**: Target under 200KB, max 500KB
- **Dimensions**: 1200-2400px wide recommended
- **Naming**: Use lowercase with hyphens (e.g., `solar-duck-curve.png`)
